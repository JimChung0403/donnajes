## 1. Design Tokens

### Colors
```css
:root {
  --color-ink: #1b1b1b;
  --color-muted: #6f6f6f;
  --color-bg: #f7f7f7;
  --color-surface: #ffffff;
  --color-surface-alt: #fdf1f3;
  --color-border: #e7e7e7;
  --color-brand: #ff385c;
  --color-brand-strong: #e31c5f;
  --color-brand-soft: rgba(255, 56, 92, 0.12);
  --color-brand-soft-2: rgba(255, 56, 92, 0.18);
}
```

### Typography
- Font Sans: `var(--font-sans)` (Noto Sans TC)
- Font Serif: `var(--font-serif)` (Noto Serif TC)

Type Scale:
- Display: 3.6rem / 700 / 1.1
- H1: 3rem / 700 / 1.2
- H2: 2.2rem / 600 / 1.3
- H3: 1.1rem / 600 / 1.4
- Body: 1rem / 400 / 1.7
- Caption: 0.85rem / 500 / 1.4

### Spacing
- `--space-1` 4px
- `--space-2` 8px
- `--space-3` 12px
- `--space-4` 16px
- `--space-5` 20px
- `--space-6` 24px
- `--space-8` 32px
- `--space-10` 40px
- `--space-12` 48px
- `--space-16` 64px

### Radius
- `--radius-sm` 12px
- `--radius-md` 18px
- `--radius-lg` 28px
- `--radius-full` 999px

### Shadow
- `--shadow-sm` 0 6px 16px rgba(17, 17, 17, 0.06)
- `--shadow-md` 0 16px 32px rgba(17, 17, 17, 0.1)
- `--shadow-lg` 0 32px 48px rgba(17, 17, 17, 0.14)

### Motion
- Default easing: `cubic-bezier(0.16, 1, 0.3, 1)`
- Duration: 200ms for hover, 500ms for modal

## 2. Component Library

### Button
Variants:
- Primary: `bg brand`, text white
- Secondary: `bg white`, border `color-border`, text `color-brand-strong`
- Ghost: `bg brand-soft`, border `rgba(255, 56, 92, 0.2)`, text `color-brand-strong`

States:
- Default: shadow `shadow-sm`
- Hover: slightly higher shadow + bg darken
- Active: translateY(2px)
- Focus: outline 3px `rgba(255, 56, 92, 0.35)`
- Disabled: opacity 0.5, cursor not-allowed
- Loading: spinner inline + text `載入中...`
- Error: border `color-brand-strong`

### Pill / Badge
- Default: bg `color-brand-soft`, text `color-brand-strong`
- Soft: bg `#f5f5f5`, text `#5a5a5a`
- Ghost: bg `rgba(0,0,0,0.04)`, text `color-muted`

States:
- Hover: bg darken 8%
- Active: scale 0.98
- Disabled: opacity 0.5
- Error: bg `#fff5f6`, border `rgba(255, 56, 92, 0.4)`

### Card
- Surface: `color-surface`
- Border: `color-border`
- Shadow: `shadow-sm`
- Padding: `space-5`

States:
- Hover: raise 2px, border `rgba(255, 56, 92, 0.3)`
- Active: keep raised, highlight border `color-brand`
- Focus: outline `rgba(255, 56, 92, 0.35)`
- Disabled: opacity 0.6
- Loading: skeleton blocks
- Empty: show empty state copy
- Error: border `color-brand-strong`

### Option (Question Choice)
- Base: rounded `radius-md`, bg `#f5f5f5`
- Selected: border `color-brand-strong`, bg `color-brand-soft`

States:
- Hover: bg `#f0f0f0`
- Active: bg `color-brand-soft-2`
- Focus: outline `rgba(255, 56, 92, 0.35)`
- Disabled: opacity 0.5
- Loading: skeleton line
- Error: border `color-brand-strong`

### Modal / Overlay
- Overlay: `rgba(17, 17, 17, 0.5)`
- Modal: radius `radius-lg`, shadow `shadow-lg`

States:
- Open: pop in animation
- Loading: spinner + copy
- Error: warning text + retry CTA

## 3. Page Layouts

### Topbar
- Left: brand mark + sublabel
- Center: 3 anchor links
- Right: ghost CTA button

### Hero
- Left: badges, H1, lead copy, 2 CTA buttons, helper note
- Right: hero panel card + stats + preview cards

### Product Section
- Header row with section title + design chip
- Grid cards with image, tone pill, name, highlights, fit list

### Questionnaire Section
- Header with title + step list
- Grid of question cards (12)
- CTA row with primary/secondary + notice

### Result Modal
- Header + close button
- Insight pills row
- Grid of recommended cards (2)
- Feedback panel with detail tips

## 4. Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

Rules:
- Hide topbar links on < 900px
- Hero stats stack on mobile
- Grid auto-fit 250px min
