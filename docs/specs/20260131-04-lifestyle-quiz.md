# Spec: 生活型態問券推薦區塊

**Date**: 2026-01-31  
**Spec ID**: 20260131-04  
**Author**: PM  
**Status**: Review

---

## 1. Background

### Problem Statement
目前網站缺乏能引導使用者快速找到適合產品的互動入口，導購動線偏被動。

### Business Value
以 7-8 題生活型態問券提升停留時間與參與感，並產出可行動的產品推薦與詳情導向。

### Success Metrics
- [ ] 問券完成率 ≥ 35%
- [ ] 推薦卡片點擊率 ≥ 30%
- [ ] 產品詳情區塊/頁面停留時間 ≥ 20% 提升

---

## 2. User Stories

### Primary User Story
As a 25-45 歲女性使用者，  
I want 用簡單問券快速理解適合我的產品，  
So that 我能更有信心查看詳情並做購買決策。

### Secondary Stories
- As a 使用者，我想看見推薦原因而不是只有產品名稱。  
- As a 使用者，我希望點擊推薦後直接看到產品詳情。  

---

## 3. Acceptance Criteria

### Happy Path
- [ ] Given 使用者完成 7-8 題單選題, when 提交, then 顯示 Top 2 推薦產品與推薦原因。  
- [ ] Given 推薦結果, when 點擊 CTA, then 導向該產品詳情（頁面區塊或專頁）。  

### Edge Cases
- [ ] Given 使用者未完成全部題目, when 提交, then 顯示提示並標示未完成題目。  
- [ ] Given 分數相同, when 產生推薦, then 依既定優先序或隨機補足。  

### Error Handling
- [ ] Given 任何題目無法載入, when 顯示問券, then 提示錯誤與重試按鈕。

## 5. Recommendation Logic (PM 決定)

### 產品池
- 婕樂纖【厚焙奶茶】（輕卡營養）
- 爆纖錠（代謝活力）
- 我的祕密 纖飄錠（體態管理）
- 植萃纖酵宿（腸道順暢）
- 高機能益生菌（日常保養）
- 雪花紫纖飲（清爽代謝）

### 問題設計（單選 7-8 題）
1. 作息時間  
2. 運動頻率  
3. 外食頻率  
4. 含糖飲料/甜點頻率  
5. 排便頻率  
6. 壓力程度  
7. 喝水量  
8. 久坐時間（可選擇是否採用）

### 計分邏輯（加權）
- 每題單選 → 對應 1-2 個產品 +1~3 分  
- 依總分排序取 Top 2  
- 若分數相同：依「厚焙奶茶 > 植萃纖酵宿 > 纖飄錠 > 爆纖錠 > 益生菌 > 雪花紫纖飲」補足  

### 產品對應原則（範例）
- 作息偏晚／久坐 → 爆纖錠 +2、厚焙奶茶 +1  
- 外食高／甜食高 → 纖飄錠 +2、厚焙奶茶 +1  
- 排便不規律／喝水少 → 植萃纖酵宿 +2、厚焙奶茶 +1  
- 壓力高 → 益生菌 +2  
- 想取代含糖飲料 → 雪花紫纖飲 +2、厚焙奶茶 +1
