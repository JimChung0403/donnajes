# Design Specification - Host + Experience + Reviews

**Date**: 2026-01-31
**Spec ID**: 20260131-02
**Author**: UI/UX Design
**Tech Stack**: Next.js + CSS (custom properties)

---

## 1. Additions to Tokens

- Topbar background: `rgba(255,255,255,0.92)` + border `rgba(17,17,17,0.06)`
- Host accent card: `#fff5f7` optional for About notes

---

## 2. New Components

### Host Card
Layout:
- 2-column (photo + content), mobile stacks
- Card: surface + border + shadow-md

States:
- Default: shadow-md
- Hover: raise 2px
- Focus: outline `rgba(255, 56, 92, 0.35)`
- Disabled/Loading: opacity 0.6

### About Card
Layout:
- single column, inner tags row
- Note block with dashed border

### Experience Card
Layout:
- image on top, title + description + tags
- border uses brand-soft for airy Airbnb feel

### Review Card
Layout:
- simple quote card with gradient background
- city tag as caption

---

## 3. Page Layout Updates

- Topbar: make nav links distinct from main background
- Insert new sections after hero:
  - Host (About Donna)
  - Experiences (3 cards)
  - Reviews (3 cards)

---

## 4. Responsive Rules
- Host card stacks under 900px
- Review cards in grid auto-fit 250px
