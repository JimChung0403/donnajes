# Product Manual

**Last Updated**: 2026-01-31  
**Version**: 0.1.0

---

## Overview

婕樂纖健康食品品牌官網，提供產品介紹與購買導流。

---

## Features

### 1. 首頁

**Description**: 品牌與主打產品展示，導向產品與購買流程。

**User Flow**:
1. 使用者進入首頁
2. 瀏覽品牌與主打內容
3. 點擊進入產品頁或購買

**Technical Notes**:
- API: N/A
- Data: N/A

---

### 2. 產品列表

**Description**: 列出可購買的產品（現況未明確分類/數量）。

**User Flow**:
1. 使用者進入產品列表
2. 瀏覽產品摘要
3. 點擊進入產品詳情

**Technical Notes**:
- API: N/A
- Data: N/A

---

### 3. 產品詳情

**Description**: 顯示產品資訊與購買入口。

**User Flow**:
1. 使用者進入產品詳情
2. 了解產品資訊
3. 點擊加入購物車或結帳

**Technical Notes**:
- API: N/A
- Data: N/A

---

### 4. 結帳流程

**Description**: 完成訂購與付款。

**User Flow**:
1. 使用者從產品頁進入結帳
2. 填寫資料/付款
3. 完成訂單

**Technical Notes**:
- API: N/A
- Data: N/A

### 5. 內容頁（品牌故事／健康知識）

**Description**: 品牌與知識內容頁。

**User Flow**:
1. 使用者瀏覽內容頁
2. 了解品牌與健康資訊
